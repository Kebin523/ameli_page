OVERVIEW

아멜리는 '화장이 처음인 누군가에게도 메이크업 아티스트에게도 아멜리는 쉽습니다' 라는 브랜드 슬로건을 가진 컬러 코스메틱 브랜드다. 기존 아멜리의 메인 웹사이트는 풀스크린 배너와 세로형 사이드 메뉴로 구성되어있었다. 기존의 형태를 유지하면서 브랜드 슬로건처럼 누구나 쉬울 수 있도록 브랜드의 메인 컨텐츠를 확인 가능한 메인 사이트 디자인을 제안한다.

AS IS

풀페이지 배너만으로 구성된 메인페이지

사이드 메뉴 중 아코디언 메뉴 구분 어려움

아코디언 메뉴 활성화 시 하단 메뉴가 가려지는 현상 발생

TO BE

배너와 더불어 아멜리의 주요 컨텐츠에 대한 영역을 추가하여 접근성을 높인다.

아코디언 메뉴의 경우 아이콘을 추가하여 일반 메뉴와 구분한다.

사이드 메뉴 영역 안에 스크롤 기능을 추가하여 아코디언 메뉴가 활성화되어도 하단 메뉴를 확인할 수 있도록 고안한다.

기존 웹사이트
